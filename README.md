# 🌶️ Joie-Épice

> Une application web e-commerce réalisée avec **Symfony**.

> ## 🚀 Lancer le projet en local
> ### 1. Cloner le dépôt
git clone https://github.com/DKanthio/Joie-epice.git
cd Joie-epice
2. Installer les dépendances:
composer install

3. Configuration de l’environnement
Dans le fichier .env.local à la racine du projet , ajoute la ligne suivante (⚠️ clé secrète non versionnée) :
MAILER_DSN=sendgrid://SG.lqkfaCTcQRacOxaBCR6V1g.LusRci9aP1Ic6fspYQAHGYvY4iKC35my0NpQBk

5. Lancer le serveur
symfony server:start
L’application sera accessible à l’adresse :

🔗 https://127.0.0.1:8000/

🧪 Tests
Les tests sont disponibles dans le répertoire tests/.

Pour les exécuter :

php bin/phpunit
